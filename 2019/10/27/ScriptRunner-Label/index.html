ScriptRunner Label лӢӨлЈЁкё°

мҠӨнҒ¬лҰҪнҠёлҹ¬л„Ҳм—җм„ң Jira мқҙмҠҲмқҳ Label к°’мқ„ мҲҳм •н•ҳлҠ” кІҪмҡ°к°Җ мў…мў… мһҲлҠ”лҚ° м–ҙл–»кІҢ м¶”к°Җ/мҲҳм •н•ҳлҠ”м§Җ
мқҙлІҲ нҸ¬мҠӨнҠём—җм„ң лӢӨлӨ„ліҙкІ мҠөлӢҲлӢӨ.

мқҙлІҲм—җлҸ„ Script Consoleм—җм„ң мҪ”л“ңлҘј мӢӨн–үмӢңмјңліҙкі  м ҖлҠ” м–ҙл–Ө мјҖмқҙмҠӨм—җм„ң мӮ¬мҡ©н•ҳлҠ”м§Җ м„ӨлӘ…л“ңлҰ¬кІ мҠөлӢҲлӢӨ.

мӢңмҠӨн…ң н…ҢмҠӨнҠё нҷҳкІҪ

мҠӨнҒ¬лҰҪнҠёлҹ¬л„ҲлҠ” Jira лІ„м „м—җ мҳҒн–Ҙмқ„ мўҖ л°ӣм•„м„ң нҷҳкІҪлҸ„ лҜёлҰ¬ м•Ңкі  кі„м…”м•јн•©лӢҲлӢӨ.
(лІ„м „м—җ л”°лқј лӮҙл¶Җ н•ЁмҲҳл“Өмқҙ ліҖкІҪлҗЁм—җ л”°лқј мӢӨм ң мҪ”л“ңлҸ„ мЎ°кёҲм”© ліҖкІҪлҗ©лӢҲлӢӨ)

  • Jira: Jira Software 8.3.2
  • ScriptRunner: 5.6.1.1-jira8

Label к°’ к°Җм ёмҳӨкё°

Label к°’мқ„ к°Җм ёмҳӨлҠ” кІғмқҖ мүҪмҠөлӢҲлӢӨ.
мқҙмҠҲм—җм„ң кё°ліём ҒмңјлЎң мӮ¬мҡ©н•ҳлҠ” н•„л“ңлқјм„ң лӢӨлЈЁкё° мү¬мҡҙ кІғмқҙ мһҘм җмһ…лӢҲлӢӨ.

getLabels(Long issueId)

getLabels() н•ЁмҲҳлҠ” issueId к°’мқ„ л°ӣм•„ мқҙмҠҲмқҳ label к°’мқ„ к°Җм ёмҳөлӢҲлӢӨ.
н•ЁмҲҳмқҳ лҰ¬н„ҙ к°’мқҖ labelл“Өмқ„ нҸ¬н•Ён•ң л°°м—ҙлЎң лҰ¬н„ҙн•©лӢҲлӢӨ.
(м•„лһҳ кІ°кіј к°’мқҖ мқҙмҠҲм—җ н…ҢмҠӨнҠёлЎң Label к°’мқ„ мһ…л Ҙн•ң кІғмқ„ кё°л°ҳмңјлЎң мҠӨнҒ¬лҰҪнҠё кІ°кіјк°Җ лӮҳмҳЁ кІғмһ…лӢҲлӢӨ.)

Label к°’мқ„ м¶”к°Җн•  л•Ңм—җлҠ” м–ҙл–Ө мӮ¬мҡ©мһҗк°Җ к°’мқ„ м¶”к°Җн•  кІғмқём§Җ кІ°м •мқҙ н•„мҡ”н•©лӢҲлӢӨ.
лЁјм Җ мҪ”л“ңлҘј ліҙл©ҙм„ң м„ӨлӘ…мқ„ л“ңлҰ¬кІ мҠөлӢҲлӢӨ.

мҪ”л“ң м„ӨлӘ…

м•һм„ң Label к°’мқ„ к°Җм ёмҳӨлҠ” кІғкіј л№„көҗн•ҙм„ң лӢ¬лқјм§„ кІғмқҖ м•„лһҳ мҪ”л“ңмҷҖ addLabel() н•ЁмҲҳ мһ…лӢҲлӢӨ.
def user = ComponentAccessor.getUserManager().getUserByKey("pubg-helper");
мң„ мҪ”л“ңлҠ” Labelмқ„ м¶”к°Җн•  мӮ¬мҡ©мһҗлҘј кІ°м •н•ҳкё° мң„н•ҙ мӮ¬мҡ©мһҗ к°қмІҙлҘј м•„мқҙл””лЎң к°Җм ёмҳЁ кІғмһ…лӢҲлӢӨ.

ScriptRunnerмқҳ Listener кё°лҠҘм—җм„ң Label мҲҳм •/м¶”к°Җ кё°лҠҘмқ„ мӮ¬мҡ©н•  л•Ң,
мң„ мҪ”л“ңлҘј нҶөн•ҙ м—…лҚ°мқҙнҠён•ҳлҠ” мӮ¬мҡ©мһҗлҘј лҸҷм ҒмңјлЎң ліҖкІҪн•  мҲҳ мһҲмҠөлӢҲлӢӨ.

addLabel(User remoteUser, Long issueId, String label, boolean sendNotification) н•ЁмҲҳлҠ” мқҙмҠҲм—җ Labelмқ„ н•ҳлӮҳ м¶”к°Җн•  мҲҳ мһҲлҠ” н•ЁмҲҳмһ…лӢҲлӢӨ.

labelManager.setLabels()

labelManager.setLabels() н•ЁмҲҳлҠ” Label к°’мқ„ м¶”к°Җ, мӮӯм ңн•ңлӢӨкё°ліҙлӢӨлҠ”

к·ёлһҳм„ң as SetмңјлЎң ліҖкІҪн•ҙмӨҖ кІғмңјлЎң мқҙн•ҙн•ҙмЈјмӢңл©ҙ лҗ©лӢҲлӢӨ. (нҳ№мқҖ ліҖмҲҳлҘј л”°лЎң SetмңјлЎң л‘җкі  мһ…л Ҙн•ҳл©ҙ лҗ©лӢҲлӢӨ)
